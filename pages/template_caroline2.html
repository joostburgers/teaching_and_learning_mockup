DIGITAL Yoknapatawpha · Teaching & Learning

Preface

Following reading William Faulkner's, "Lion", use the following tools to evaluate and answer the provided response questions to assess how the changes within the story push a bigger fear on the way the town is changing.

Tip

This module can be taught by itself or in conjunction with any stories that are part of the Faulkner and modernization cluster:

Activities

1. First Activity- Mapping Lions stomping grounds

Explore: Narrative Analysis

Overview

Using the mapping tool users are able to see the locations in which the story takes place as well as the characters location on the map throughout the narrative. By using this tool users are able to click on locations and characters to get further information and descriptions of their importance to help to further allow analysis of the text.

Walkthrough: About the text tool

The map display allows users to pull up records for every location, character, and event in the story. These records contain an overview of each character, the events where they are present and mentioned, and the locations where those events take place.

Instructions
  1. From the Digital Yoknapatawpha homepage, select “Lion” from the bookshelf.
  2. Hover over the map icons until you find “Holly Knoll’” ()
  3. Click on the icon
    • A location record will pop up in a new window
  4. Read the description of the location
  5. Repeat these steps for:
    • “Boon Hogganbeck” ()
    • “ Major de Spain”()

Using the About the Text section, use your best interpretation of the information to answer the following questions.

  • Where are the majority of the characters and events located in this text?
  • Boon Hogganbeck- how does Boon being perceived as mixed race affect how his actions are interpreted by the others?
  • Holly Knoll- how might the imagery attached to Lion’s burial ground help further the masculine norms in the narrative?
  • Major de Spain- How might the fact that he is able to escape to his mundane job allow him to move on from Lions death, unlike Boon Hogganbeck?
Instructions
  1. From the DY Main Menu choose:
    • Visualizations
    • Narrative Analysis
  2. Select “Lion” from the drop-down menu on the left.
  3. Hover over each chart to see more detailed information about each data point
  4. Click “Add Chart”
  5. Select “Lion” from the new drop-down menu
  6. Click “Date Order” on one of the charts to compare the different time structures

2. Second Activity- Correlation Between Connections

Explore: Character Connections

Overview

Using the Character-Character graph users are able to examine the connection each character has with one another, while also limiting the search to the certain personal traits to evaluate the characters from a new perspective. Using the graph students should be able to physically see stronger connections and should be able to answer questions and evaluate the narrative with this new lens in mind.

Walkthrough: Character-Character Force Directed Graph

This graph shows how often characters are present or mentioned in the same event. A red line () between two blue boxes () shows at least one interaction, and a thicker line indicates multiple interactions. Characters with more connections are closer to the center, but, importantly, the position of characters are not fixed.

Characters who appear close together in one graph may appear far apart in another. The graph is interactive, so you can click on a character’s name to see their connections. You can also click and drag characters to reposition them.

Instructions
  1. Click "Visualization" on the navigation bar at the top of the Digital Yoknapatawpha homepage
    • Click Character-Character
  2. Set Text as “Lion”
  3. Complete the following searches and answer the related questions:
    1. All characters
      • Set Text as “Lion”
      • Leave all other fields blank
      • Click Search
    2. Race
      • Set Race as “White”
      • Click Search
      • Repeat for each race
      • Switch Rank between “Major” and “Major and Secondary”
    3. Gender
      • Set Gender as “Male”
      • Click Search
      • Repeat for each gender
    4. Class
      • Set Class as “Upper Class”
      • Click Search
      • Repeat for each class
    5. Other
      • Select any remaining variables
      • Click Search

        Use the Location-Character Graph to help visual and elaborate on the following questions.

        • What is the race of the main three characters?
        • How does the ration of women to men in this narrative allow this story to exist?
        • How does the role of masculinity get portrayed in the narrative?
        • How does the different class perspective of the character allow them to view the death of Lion?

3. Third Activity- Context on the Text

Explore: Character Search

Overview

Using the About the Text tool users are able to get further background information on the texts. By accessing this information users are able to expand on the importance and significance of the specific narrative that they have read.

Instructions
  1. Navigate to the Digital Yoknapatawpha home page
  2. Select “Lion” from the bookshelf
    • This will open the main map display for “Lion”
  3. After selecting Lion, the page will appear with the map, display control panel, title with the contributing authors, timeline, and page correlation.
  4. Select the title with the contributing author in the left bottom corner, and the About the Text will appear.
  5. Select a character for your fan fic by clicking on it
    • For a more targeted search, check the “Alphabetical List” box under “Show Characters”
    • Check the name of a specific character
    • Click “Show”

Using the map, work together in groups to answer the following questions with quotes to support your claims. Present your answers and evidence to the class.

  • Approximately how many different versions of Lion are there?
  • Throughout Faulkner's rewrites, what is it important about reading a version where the character does not specifically see the climactic scene instead reports the version he has heard?
  • How can the created environment of text lead to the characters being set up to fail?
